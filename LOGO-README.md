# 慧界极简 LOGO 使用指南

## 📁 LOGO 文件说明

本品牌提供4个版本的LOGO文件，全部为SVG矢量格式，可无损缩放：

### 1. **logo.svg** - 完整版LOGO
- **尺寸**: 280 × 100 px
- **内容**: 图标 + 中文名称 + 英文副标题
- **适用场景**:
  - 网站页眉/页脚
  - 官方文档
  - 名片
  - 宣传资料
  - PPT演示

### 2. **logo-light.svg** - 浅色背景版
- **尺寸**: 280 × 100 px
- **内容**: 图标（彩色）+ 深色文字
- **适用场景**:
  - 白色/浅色背景
  - 打印材料
  - PDF文档
  - 浅色网页

### 3. **logo-icon.svg** - 图标版
- **尺寸**: 100 × 100 px
- **内容**: 仅智慧之光图标
- **适用场景**:
  - 社交媒体头像
  - App图标
  - 小尺寸展示
  - 水印
  - 微信公众号头像

### 4. **favicon.svg** - 网站图标
- **尺寸**: 100 × 100 px
- **内容**: 白色图标 + 紫色渐变背景
- **适用场景**:
  - 浏览器标签页图标
  - 收藏夹图标
  - PWA应用图标
  - 已自动集成到网站

## 🎨 设计理念

### 智慧之光
- **中心圆点**: 象征智慧的源泉，代表"慧"字的核心含义
- **放射光芒**: 8条对称光线，代表智慧的传播与影响力
- **极简美学**: 简洁几何图形，体现"极简"设计哲学
- **科技渐变**: 品牌紫色渐变，传达现代科技感

### 品牌色彩
- **主色调**: #667eea (紫蓝色)
- **辅助色**: #764ba2 (深紫色)
- **渐变**: linear-gradient(135deg, #667eea 0%, #764ba2 100%)

## 💻 使用方法

### 在网站中使用

```html
<!-- 完整LOGO -->
<img src="/logo.svg" alt="慧界极简" width="280" height="100">

<!-- 浅色背景版 -->
<img src="/logo-light.svg" alt="慧界极简" width="280" height="100">

<!-- 图标版 -->
<img src="/logo-icon.svg" alt="慧界极简" width="100" height="100">
```

### 在React组件中使用

```jsx
// 导入
import logo from '/logo.svg'
import logoLight from '/logo-light.svg'
import logoIcon from '/logo-icon.svg'

// 使用
<img src={logo} alt="慧界极简" className="header-logo" />
```

### 作为CSS背景

```css
.header {
  background-image: url('/logo.svg');
  background-size: contain;
  background-repeat: no-repeat;
}
```

## 📏 尺寸规范

### 最小使用尺寸
- **完整LOGO**: 最小宽度 140px
- **图标LOGO**: 最小尺寸 24 × 24 px

### 推荐尺寸
- **网站头部**: 200-280px 宽
- **社交媒体头像**: 512 × 512 px
- **名片**: 根据实际尺寸等比缩放
- **打印材料**: 300 DPI 输出

### 安全区域
LOGO周围应保持至少等于图标高度25%的空白区域，确保视觉清晰。

## 🎯 使用规范

### ✅ 正确使用

1. **保持完整性**: 不要拆分或重组LOGO元素
2. **保持比例**: 等比例缩放，不要拉伸变形
3. **保持颜色**: 使用官方色彩，不要随意改色
4. **保持清晰**: 确保LOGO清晰可见，不要模糊或像素化
5. **保持对比**: 选择合适的背景，确保LOGO清晰可辨

### ❌ 禁止行为

1. ❌ 改变LOGO颜色（除非使用指定的浅色版）
2. ❌ 旋转、倾斜或扭曲LOGO
3. ❌ 添加阴影、描边或其他效果
4. ❌ 将LOGO放在复杂背景上导致难以识别
5. ❌ 修改LOGO的任何元素
6. ❌ 用其他字体替换文字部分
